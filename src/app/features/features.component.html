Features

Trackers: Design the flexible data inputs yourself

You can design your own data format by configuring various data types such as number, text, time point, time range, choices, location, or audio records. You can also change the configuration even if you've already been tracking data.

The configuration UI is user-friendly, similar to that of form-building system such as Google Forms or Survey Monkey.

Services: Integrate external trackers and other services

You can connect the external services to OmniTrack's data field. For example, you can connect Fitbit step count to a number field.
